# Checklist QA - Paridade com PGR Marina

## Visual
- [ ] Fonte visualmente equivalente ao template real.
- [ ] Pesos tipográficos corretos em títulos, subtítulos e células.
- [ ] Espaçamento vertical consistente entre seções.
- [ ] Bordas e grid das tabelas sem desalinhamento.
- [ ] Numeração de páginas e quebras sem corte de conteúdo.

## Conteúdo
- [ ] Seções 1-5 corretas para o card de referência.
- [ ] Seções estáticas (5-18) sem alteração indevida.
- [ ] Seção 19+ legível em todos os GHEs do caso Marina.
- [ ] Campo ANL com valor esperado pelo histórico.
- [ ] Responsáveis corretos (elaboração, coordenação, implementação).

## Casos de Volume
- [ ] 1 GHE e poucos riscos.
- [ ] 10+ GHEs e muitos riscos.
- [ ] Descrição de atividade longa.
- [ ] EPC/EPI com listas extensas.
- [ ] Plano de ação com múltiplas páginas.

## Consistência de Dados
- [ ] O PDF reflete exatamente o estado salvo no backend.
- [ ] Regerar sem alterar estado produz resultado equivalente.
- [ ] Datas no índice de anexos respeitam presença real de dados.
- [ ] Não há preenchimento com dados de outro card/empresa.

## Aceite
- [ ] Aprovado por comparação lado a lado com o Marina real.
- [ ] Diferenças remanescentes documentadas com justificativa.
